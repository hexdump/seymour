# seymour

*Feed me training data, Seymour!*

A genetic algorithm-based, non-standard neural network solver library.

## Getting Started

### Installation

```
$ cargo install

```

Then, use the following code:

```rust
use seymour::solve();

fn main() {
  ...
  solve(vec![2, 3, 4, 1], &mut dataset);
}

And seymour will work its magic!
```
